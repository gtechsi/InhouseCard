Instruções para Criar Índices no Firestore

Para criar os índices necessários para a aplicação, acesse o console do Firebase:

https://console.firebase.google.com/project/inhousecard-a17c9/firestore/indexes

Você precisa criar os seguintes índices compostos:

1) Coleção: "orders"

Campos:

  • user_id (Ascending)
  • status (Ascending)
  • created_at (Descending)

2) Coleção: "orders"

Campos:

  • user_id (Ascending)
  • created_at (Descending)

3) Coleção: "orders"

Campos:

  • status (Ascending)
  • created_at (Descending)

IMPORTANTE:

Após criar os índices, aguarde alguns minutos para que eles sejam construídos.

Então tente acessar a página de pedidos novamente na aplicação.
